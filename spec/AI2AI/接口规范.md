## 对话接口

### POST /api/chat/stream
流式对话接口，使用 Server-Sent Events (SSE)。

**请求体**:
```json
{
  "message": "用户消息内容",
  "session_id": "会话标识（可选）",
  "stream": true
}
```

**响应格式**: `text/event-stream`

**事件类型**:
```
data: {"type": "start", "session_id": "xxx"}

data: {"type": "content", "content": "文本块"}

data: {"type": "summary", "message": "对话已自动总结"}

data: {"type": "done", "session_id": "xxx"}

data: {"type": "error", "message": "错误信息"}
```

**事件字段说明**:
| type | 说明 | 附加字段 |
|------|------|---------|
| start | 流式开始 | session_id |
| content | 内容块 | content |
| summary | 对话总结 | message |
| done | 流式结束 | session_id |
| error | 错误信息 | message |

### POST /api/chat/message
非流式对话接口，返回完整响应。

**请求体**: 同 `/api/chat/stream`

**响应**:
```json
{
  "session_id": "会话标识",
  "response": "回复内容",
  "summarized": false
}
```

## 会话管理

### GET /api/session/{session_id}
获取对话历史。

**响应**:
```json
{
  "session_id": "会话标识",
  "messages": [
    {"role": "user", "content": "用户消息", "timestamp": "..."},
    {"role": "assistant", "content": "助手回复", "timestamp": "..."}
  ]
}
```

### DELETE /api/session/{session_id}
清空对话历史。

**响应**:
```json
{
  "message": "会话 xxx 已清空"
}
```

### POST /api/session/{session_id}/summarize
手动触发对话总结。

**响应**:
```json
{
  "session_id": "xxx",
  "summary": "对话摘要内容",
  "summarized": true
}
```

## SSE 客户端示例

### JavaScript
```javascript
async function chat(message, sessionId) {
  const response = await fetch('/api/chat/stream', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      message: message,
      session_id: sessionId
    })
  });

  const reader = response.body.getReader();
  const decoder = new TextDecoder();

  while (true) {
    const { done, value } = await reader.read();
    if (done) break;

    const text = decoder.decode(value);
    const lines = text.split('\n');

    for (const line of lines) {
      if (line.startsWith('data: ')) {
        const data = JSON.parse(line.slice(6));

        switch (data.type) {
          case 'start':
            console.log('Session:', data.session_id);
            break;
          case 'content':
            process.stdout.write(data.content);
            break;
          case 'done':
            console.log('\n完成');
            break;
          case 'error':
            console.error('错误:', data.message);
            break;
        }
      }
    }
  }
}
```
