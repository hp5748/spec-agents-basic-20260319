# 调用链追踪模块说明

## 模块概述

**模块名称**: ChainTracker
**位置**: `src/agent/chain_tracker.py`
**用途**: 追踪并格式化 Agent 调用链信息

## 核心类

### ChainInfo

```python
@dataclass
class ChainInfo:
    """调用链节点信息"""
    source_type: str   # "skill", "subagent", "mcp", "llm"
    source_name: str   # 模块名称
    confidence: float  # 匹配置信度 0.0-1.0

    def format(self) -> str:
        """格式化为显示文本"""
        # 返回: "Skill: xxx", "SubAgent: yyy", "MCP: zzz", "LLM"
```

### ChainTracker

```python
class ChainTracker:
    """调用链追踪器"""

    def __init__(self):
        self._chain: List[ChainInfo] = []

    def add(self, source_type: str, source_name: str, confidence: float = 0.0):
        """添加调用链节点"""

    def format_signature(self) -> str:
        """格式化调用链签名
        返回: "\n\n[Skill: xxx]" 或 "\n\n[LLM]"
        """

    def clear(self):
        """清空调用链"""

    def is_empty(self) -> bool:
        """是否为空"""
```

## 集成方式

### StreamAgent 集成

**文件**: `src/agent/stream_agent.py`

```python
class StreamAgent:
    def __init__(self, ...):
        # 现有初始化...
        self._chain_tracker = ChainTracker()

    async def _try_skill(self, user_input: str) -> Optional[SkillResult]:
        # ... 意图识别和执行逻辑 ...

        if result.success:
            # 记录调用链
            self._chain_tracker.add(
                source_type="skill",
                source_name=intent_result.skill_name,
                confidence=intent_result.confidence
            )

        return result

    async def chat_stream(self, user_input: str, include_history: bool = True):
        # ... 尝试 Skill ...

        if skill_result and skill_result.success:
            # 输出响应内容
            for char in skill_result.response:
                yield char

            # 输出调用链签名
            signature = self._chain_tracker.format_signature()
            for char in signature:
                yield char

            self._chain_tracker.clear()
            return

        # LLM 直接回答
        async for chunk in self._llm_chat_stream(user_input, include_history):
            full_response += chunk
            yield chunk

        # LLM 也显示标识
        signature = self._chain_tracker.format_signature()  # 返回 [LLM]
        for char in signature:
            yield char

        self._chain_tracker.clear()
```

## 输出格式

### 显示格式

```
# Skill 调用
回答内容...

[Skill: sqlite-query-skill]

# SubAgent 调用
分析结果...

[SubAgent: code-analyzer]

# MCP 调用
文件已读取...

[MCP: filesystem]

# LLM 直接回答
你好！

[LLM]

# 复杂调用链
处理完成...

[Skill: code-analyzer → SubAgent: complexity-calculator]
```

### API 响应格式

**非流式响应**:
```json
{
  "session_id": "xxx",
  "response": "回答内容\n\n[Skill: sqlite-query-skill]"
}
```

**流式响应 (SSE)**:
```
data: {"type": "content", "content": "回答"}

data: {"type": "content", "content": "\n\n[Skill: xxx]"}

data: {"type": "done", "session_id": "xxx"}
```
