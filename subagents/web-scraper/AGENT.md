# 网页抓取 Agent

## 描述

负责网页内容提取和基础数据处理的 Agent。

## 能力

- URL 提取
- 邮箱地址提取
- 手机号提取
- 基础数据统计

## 触发关键词

- 抓取网页
- 提取 URL
- 爬虫
- scrape
- extract
- http://
- https://

## 使用示例

```
用户: 从这段文本提取信息：联系我们要发邮件到 info@example.com 或拨打 13812345678
Agent:
## 网页数据提取结果

### 发现的邮箱
- info@example.com

### 发现的手机号
- 13812345678

### 统计
- URL: 0 个
- 邮箱: 1 个
- 手机号: 1 个
```
